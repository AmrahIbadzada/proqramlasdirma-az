# 📄 CV Challenge — Proqramçı CV-ni Hazırla

> Bu sənəd proqramlaşdırma sahəsindəki CV hazırlama meyarlarını izah edir. İş müraciəti etməzdən əvvəl bu yoxlama siyahısından istifadə et.

---

## 🎯 Proqramçı CV-si Nədir?

Proqramçı CV-si standart bir CV-dən fərqlidir. İşəgötürənlər teknik bacarıqları, real layihələri, GitHub aktivliyini və kod keyfiyyətini qiymətləndirir. **"Çox iş etmişəm" deyil, "bu nəticələri əldə etmişəm" deyirsən.**

---

## ✅ CV Hazırlamazdan Əvvəl Olması Gərəkənlər

### Mütləq Vacib
- [ ] **GitHub hesabı** — ən azı 3-5 tamamlanmış layihə
- [ ] Her layihədə **README.md** — nə edir, hansı texnologiya, canlı link
- [ ] **Portfolio saytı** (optional amma çox tövsiyə edilir)
- [ ] **LinkedIn profili** — GitHub-a link ver

### Yaxşı Olar
- [ ] **Canlı demo linklər** (Netlify, Vercel, Render — hamısı pulsuz)
- [ ] Ən azı bir **açıq mənbə töhfəsi** (PR, issue)
- [ ] **Texniki blog** (Dev.to, Hashnode, Medium — pulsuz)

## 📝 CV Strukturu — Proqramçı üçün

### 1. Başlıq / Contact

```
Ad Soyad
📧 email@gmail.com  |  🔗 github.com/adın  |  💼 linkedin.com/in/adın
🌐 portfolio-siten.com  |  📍 Bakı, Azərbaycan
```

**Nə yazma:**
- ❌ Şəkil (Qərb şirkətləri üçün qoyma)
- ❌ Doğum tarixi, nikah statusu
- ❌ Ev ünvanının tam detalları

---

### 2. Texniki Bacarıqlar (Skills)

Dürüst ol — yalnız həqiqətən bildiklərini yaz.

**Format nümunəsi:**
```
Dillər:        Python, JavaScript (TypeScript), Java
Frontend:      React, HTML5, CSS3, Tailwind CSS
Backend:       Node.js (Express), FastAPI, Django
Verilənlər:    PostgreSQL, MongoDB, Redis, Firebase
DevOps:        Docker, Git, GitHub Actions, Linux
Bulud:         AWS (EC2, S3, Lambda), Azure Fundamentals
Alətlər:       VS Code, Postman, Figma
```

**Nə yazma:**
- ❌ "Microsoft Word" — əsas hesab edilir
- ❌ Heç bilmədiyin texnologiyalar — müsahibdə bilinəcək
- ❌ "Internet istifadəsi" — bu 2026-da CV-yə yazılmır

---

### 3. Layihələr (Projects) — Ən Önəmli Hissə

Universitetin yoxdursa BURA ən güclü hissən olmalıdır.

**Format:**
```
🚀 Layihə Adı | Tech Stack | [GitHub] [Demo]
   • Nə qurduğun — konkret, fəaliyyətyönümlü cümlələr
   • Texniki detallar — hansı problemi həll etdi
   • Nəticə/Ölçüm — mümkün olarsa rəqəm ver
```

**Yaxşı nümunə:**
```
📦 E-Ticarət API | Node.js, Express, PostgreSQL, JWT | [GitHub] [Demo]
   • 50+ məhsul üçün tam CRUD əməliyyatları ilə RESTful API qurdum
   • JWT əsaslı autentifikasiya sistemi tətbiq etdim
   • Docker ilə containerize edərək AWS EC2-yə deploy etdim
```

**Pis nümunə:**
```
E-Ticarət Saytı
   • HTML, CSS, JavaScript istifadə etdim
   • Sayt yaratdım
```

**Tövsiyə:** 3-4 güclü layihə, hər birinin canlı linki olsun.

---

### 4. İş Təcrübəsi (əgər varsa)

```
💼 Şirkət Adı — Junior Frontend Developer
   Yanvar 2024 – Bu günə qədər | Bakı, Azərbaycan
   • React ilə müştəri portalu yaratdım (10,000+ aktiv istifadəçi)
   • Page load vaxtını 40% azaltdım lazy loading tətbiq edərək
   • Agile/Scrum metodologiyası ilə 2 həftəlik sprint-lərdə işlədim
```

**Freelance / Part-time də yazıla bilər:**
```
🔧 Freelance Web Developer
   2023 – Bu günə qədər
   • 8+ müştəri üçün responsive veb sayt qurdum (Upwork)
   • Ortalama layihə müddəti: 2-3 həftə, müştəri məmnuniyyəti: 4.9/5
```

## 🔍 ATS (Applicant Tracking System) — Avtomatik Seçim Sistemi

Böyük şirkətlər CV-ləri avtomatik sistem ilə süzgəcdən keçirir. ATS-i keçmək üçün:

- **İş elanındaki açar sözləri** CV-yə əlavə et (dürüstlüklə)
- **Sadə format** — Cədvəl, rəngli qrafika, şəkil qoyma (ATS oxuya bilmir)
- **.PDF kimi göndər** — Word faylı format sındırır
- **1 səhifə** — 5 ildən az təcrübə üçün

---

## 🚩 Cover Letter (Müraciət Məktubu)

Hər şirkətə fərqli, personallaşdırılmış cover letter yaz. Şablon mesaj göndərmə.

**Struktur:**
```
1. Giriş (2 cümlə) — Hansı vəzifə üçün müraciət etdiyini və niyə bu şirkəti seçdiyini yaz
2. Dəyərin (3-4 cümlə) — Ən güclü layihən/bacarığınla nə edə bilərsən
3. Maraq (1-2 cümlə) — Şirkət haqqında konkret bir şeyi bəyəndiyini qeyd et
4. Çağırış (1 cümlə) — Müsahibə üçün hazır olduğunu bil
```

## 📊 CV Yoxlama Siyahısı

### Lazımdır
- [ ] GitHub linki (aktiv layihələrlə)
- [ ] Dürüst texniki bacarıqlar siyahısı
- [ ] 3+ tamamlanmış layihə (GitHub + canlı demo)
- [ ] Hər iş/layihə üçün nəticə yönümlü cümlələr
- [ ] .PDF formatı
- [ ] 1 səhifə (junior üçün)
- [ ] Orfoqrafiya yoxlanıb

### Yoxdursa Böyük Problem
- [ ] GitHub-da heç layihə yoxdur → **Müraciət etmə, əvvəl qur**
- [ ] Canlı demo yoxdur → **Netlify/Vercel-ə yüklə (pulsuz)**
- [ ] Şəkil var (Qərb şirkətləri) → **Çıxar**
- [ ] 3+ səhifədir (junior) → **Qısalt**
