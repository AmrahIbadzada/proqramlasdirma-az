## Veb İnkişafı (Web Development)

Daha sıx-lıqla istifadəçi interfeysi və veb saytların görünüşü üzərində çalışaraq şirkətlərin rəqəmsal varlıqlarına töhfə verən sahədir. **Frontend**, **Backend** və **Full-Stack** olaraq üç əsas qola ayrılır.

### Frontend Developer nə edir?

İstifadəçinin birbaşa gördüyü və qarşılıqlı əlaqəyə girdiyi hər şeyi qurur: düymələr, animasiyalar, formlar, mobil uyğunluq. Brauzerdə işlər.

**Əsas texnologiyalar:**
- HTML, CSS, JavaScript — bunlar olmadan olmaz
- React, Vue.js, Angular — müasir JavaScript freymvorkları
- TypeScript — JavaScript-in tip-güvənli versiyası
- CSS freymvorkları: Tailwind CSS, Bootstrap
- Build alətləri: Vite, Webpack

**Yol xəritəsi:** [roadmap.sh/frontend](https://roadmap.sh/frontend)

### Backend Developer nə edir?

Server tərəfindəki məntiqi qurur: verilənlər bazası, API-lər, autentifikasiya, biznes məntiqləri. İstifadəçi görmür amma hər şeyin çalışmasını o təmin edir.

**Əsas texnologiyalar:**
- Dillər: Python (Django/FastAPI), JavaScript (Node.js), Java (Spring Boot), Go, Rust, PHP
- Verilənlər bazası: PostgreSQL, MySQL, MongoDB, Redis
- API: REST, GraphQL
- Serverler: Nginx, Apache
- Bulud: AWS, Azure, GCP

**Yol xəritəsi:** [roadmap.sh/backend](https://roadmap.sh/backend)

### Full-Stack Developer nə edir?

Həm frontend, həm backend üzərində işləyir. Kiçik komandalar və startaplar üçün ideal. Bütün texnologiya yığınını (stack) başa düşür.

**Populyar texnologiya yığınları (stacks):**
- **MERN:** MongoDB + Express + React + Node.js
- **MEAN:** MongoDB + Express + Angular + Node.js
- **LAMP:** Linux + Apache + MySQL + PHP
- **JAMStack:** JavaScript + API + Markup

## DevOps / Konteyner

İnkişaf (Development) və Əməliyyat (Operations) dünyalarını bir araya gətirir. Kodun yazılmasından istehsalda (production) işlənməsinə qədər bütün prosesi avtomatlaşdırır.

### DevOps Nə Edir?

- **CI/CD boru kəmərləri** (pipeline) qurur — kod yazılandan servera yeritmə avtomatikdir
- İnfrastrukturu kod kimi idarə edir (Infrastructure as Code — IaC)
- Monitorinq, loglama, alert sistemlərini qurur
- Güvənilir, miqyaslana bilən sistemlər dizayn edir

### Konteyner Texnologiyaları

**Docker:**
Tətbiqi bütün asılılıqları ilə birlikdə konteyner içinə qoyur. "Mende çalışır, serverdə çalışmır" problemi yox olur.

**Kubernetes:**
Çox sayda konteyneri idarə edir. Miqyaslama, yük balansı, özünü bərpa etmə.

### Sertifikat Yolu

```
AWS CP / AZ-900  →  Docker DCA  →  CKAD  →  CKA  →  CKS
(Başlanğıc)         (Orta)        (Orta)   (Ekspert) (Ekspert)
```

| Sertifikat | Verən Qurum | Qiymət |
|-----------|------------|--------|
| Docker Certified Associate (DCA) | Mirantis | $400 |
| CKAD — Kubernetes App Developer | Linux Foundation | $395 |
| CKA — Kubernetes Administrator | Linux Foundation | $395 |
| CKS — Kubernetes Security | Linux Foundation | $395 |
| HashiCorp Certified: Terraform | HashiCorp | $70 |
| GitLab Certified CI/CD | GitLab | $650 |

**Yol xəritəsi:** [roadmap.sh/devops](https://roadmap.sh/devops)

## Mobil İnkişaf (Mobile Development)

### Android (Kotlin / Java)

Google-ın rəsmi Android inkişaf dili **Kotlin**-dir (2017-dən bəri). Java hələ də geniş istifadə olunur amma yeni layihələr üçün Kotlin tövsiyə edilir.

**Öyrənmə yolu:** Kotlin Əsasları → Android Studio → Views/Jetpack Compose → Verilənlər bazası (Room) → Şəbəkə (Retrofit) → Publish to Play Store

**Qaynaqlar:**
- [Android Developer Guides](https://developer.android.com)
- [Philipp Lackner YouTube](https://youtube.com/@PhilippLackner)

### iOS (Swift)

Apple ekosistemi üçün. Swift dili 2014-dən bəri Apple tərəfindən inkişaf etdirilir.

**Öyrənmə yolu:** Swift Əsasları → Xcode → SwiftUI / UIKit → CoreData → App Store-a Yükləmə

**Qaynaqlar:**
- [100 Days of SwiftUI](https://www.hackingwithswift.com/100/swiftui) — Pulsuz ⭐

### Cross-Platform (Flutter / React Native)

Bir kod bazasından həm Android, həm iOS tətbiqi yaratmaq.

| Texnologiya | Dil | Verən | Nə Zaman |
|------------|-----|-------|---------|
| Flutter | Dart | Google | Yüksək performanslı, gözəl UI lazımsa |
| React Native | JavaScript | Meta | JS bildiyinizdə, tez başlamaq üçün |
| Kotlin Multiplatform | Kotlin | JetBrains | Backend logic paylaşmaq üçün |

## Süni İntellekt / Maşın Öyrənməsi (AI/ML)

Proqramlaşdırmanın ən sürətlə böyüyən sahəsi. Python bu sahənin standart dilidir.

### Sahələr

**Machine Learning (ML):** Məlumat nümunələrindən öyrənən alqoritmlər (Scikit-learn)
**Deep Learning:** Sinir şəbəkələri (TensorFlow, PyTorch)
**NLP (Doğal Dil İşləmə):** Mətn analizi, LLM-lər (Transformers, LangChain)
**Computer Vision:** Şəkil tanıma (OpenCV, YOLO)
**Generative AI:** ChatGPT, DALL-E kimi modellər qurmaq (Hugging Face)

### Öyrənmə Yolu

```
Python Əsasları  →  Riyaziyyat (Xətti Cəbr, Statistika)  →
Pandas/NumPy     →  Scikit-learn (klassik ML)  →
Deep Learning (PyTorch/TensorFlow)  →  İxtisaslaşma
```

## Agile / Layihə İdarəetmə

Proqramlaşdırma komandasında işləmək üçün bu metodologiyaları bilmək lazımdır.

### Scrum

Ən populyar Agile freymvorkudur. 2-4 həftəlik sprint-lər ilə işləyir.

**Rollar:** Product Owner + Scrum Master + Development Team
**Hadisələr:** Sprint Planning → Daily Standup → Sprint Review → Retrospective

**Sertifikatlar:**
- Certified ScrumMaster (CSM) — Scrum Alliance — ~$195-500
- Professional Scrum Master (PSM I) — Scrum.org — $150

### Kanban

Görsel iş axını idarəetmə metodologiyası. Scrum qədər strukturlu deyil.

### DevOps + Agile

- **SAFe (Scaled Agile Framework)** — Böyük şirkətlər üçün
- **LeSS (Large-Scale Scrum)** — Çox komandalar üçün
- **Kanban Professional** — Kanban University sertifikatı
